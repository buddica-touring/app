📖 シーズナル定義 方法論

BUDDICA TOURING 高松店のシーズナル区分（A繁忙 / B通常 / C閑散）は、以下の4要素を加重平均した総合スコア（0-10点）で機械的に判定する。エリア固有のイベント・観光統計・レンタカー業界動向・連休カレンダーをすべて織り込む設計。

合計スコア = (E × wE + T × wT + R × wR + H × wH) / (wE + wT + wR + wH)
ティア = 合計スコア ≥ 閾値A → A / ≥ 閾値B → B / その他 → C

⑤ 高松エリア固有の調整事項

インバウンド需要

高松空港の韓国/台湾/香港/中国便利用客は日本の祝日カレンダーと無関係に動く。2-3月の旧正月・10月の中国国慶節等は別途係数で調整。スコアR（レンタカー需要）に+1〜+2 加算する運用を推奨。

2025年瀬戸内国際芸術祭の反動

2025年は過去最高需要 → 2026年は前年同月比減のリスク。次回開催は2028年。2026年初年度は閾値Aを 6.5 → 7.0 に上げる慎重設定も検討。

新店舗の認知度ペナルティ

初年度は OTA 掲載順位後方。スコアではなく基本料金を中央値の70-75%に設定して認知拡大優先。

長雨・台風の即応調整

6月梅雨末期・9月台風期は天候により実需要が大きくぶれる。スコアTの予報補正（▲2点）を運用で適用可能。

⑥ 運用フロー

  1. 本書の方法論で2026/1/1 - 2027/12/31 の730日に対し、4要素スコア → 合計スコア → ティアを自動算出
  2. 基本料金（タイプA〜F のB日基準）を OTA市場調査結果から決定
  3. ティア係数（A:1.50 / B:1.00 / C:0.80）を適用して日別価格を生成
  4. OTAカレンダーに反映（自動連携 or 手動投入）
  5. 月次レビューで実需要 vs 想定の乖離を確認し、4要素重み・閾値を四半期単位で見直し
